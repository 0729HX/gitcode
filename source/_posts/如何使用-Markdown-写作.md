## 前言

Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档，然后转换成有效的 HTML 文档。本文将介绍 Markdown 的基本语法和常用技巧。

## 标题

Markdown 支持六级标题，使用 `#` 符号表示：

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

## 文本格式

### 强调文本

- **粗体**：使用 `**文本**` 或 `__文本__`
- *斜体*：使用 `*文本*` 或 `_文本_`
- ***粗斜体***：使用 `***文本***` 或 `___文本___`

### 删除线

~~删除的文本~~ 使用 `~~文本~~`

### 下划线

<u>下划线文本</u> 使用 HTML 标签 `<u>文本</u>`

## 列表

### 无序列表

使用 `-`、`+` 或 `*` 开头：

```markdown
- 项目一
- 项目二
  - 子项目 A
  - 子项目 B
- 项目三
```

### 有序列表

使用数字加句点：

```markdown
1. 第一步
2. 第二步
3. 第三步
```

## 链接和图片

### 链接

```markdown
[链接文本](https://example.com)
[带标题的链接](https://example.com "鼠标悬停显示的标题")
```

### 图片

```markdown
![图片描述](https://example.com/image.jpg)
![带标题的图片](https://example.com/image.jpg "图片标题")
```

## 代码

### 行内代码

使用反引号 \` 包裹代码：`const x = 1;`

### 代码块

使用三个反引号包裹代码，可以指定语言：

\`\`\`javascript
function hello() {
  console.log("Hello, World!");
}
\`\`\`

## 引用

使用 `>` 符号：

> 这是一段引用文本
>
> > 这是嵌套引用

## 表格

使用 `|` 分隔列，`-` 分隔表头和内容：

```markdown
| 姓名 | 年龄 | 职业 |
|------|------|------|
| 张三 | 25   | 工程师 |
| 李四 | 30   | 设计师 |
```

## 分隔线

使用三个或更多的 `-`、`*` 或 `_`：

```markdown
---
***
___
```

## 任务列表

```markdown
- [x] 已完成的任务
- [ ] 未完成的任务
- [ ] 另一个未完成的任务
```

## 转义字符

如果需要显示 Markdown 特殊字符，使用反斜杠 `\` 转义：

```markdown
\*不是斜体\*
\# 不是标题
```

## HTML 标签

Markdown 支持直接使用 HTML 标签：

```html
<div style="color: red;">红色文本</div>
<br>
<p>段落</p>
```

## 最佳实践

1. **保持简洁**：Markdown 的优势在于简洁，避免过度使用复杂的 HTML
2. **一致性**：在整个文档中保持一致的格式风格
3. **可读性**：即使在纯文本格式下，也要确保文档易于阅读
4. **预览**：使用支持 Markdown 的编辑器实时预览效果

## 推荐工具

- **编辑器**：Typora, VS Code, Mark Text
- **在线工具**：StackEdit, Dillinger
- **浏览器插件**：Markdown Viewer

## 总结

Markdown 是一种简单而强大的写作工具，掌握基本语法后，可以快速创建格式良好的文档。通过不断练习，您将能够熟练运用 Markdown 进行各种文档写作。

开始使用 Markdown，享受高效的写作体验吧！
